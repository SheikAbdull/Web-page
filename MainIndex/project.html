MACHINE LEARNING-BASED RECOMMENDATION SYSTEM FOR ONLINE MARKETS.

Nowadays online shopping is quite like a culture.Everyone tends to spend their time shopping online. From grocery to fresh vegetables to clothes and footwears,people at any ages would like to buy them online. The system aims to help firms target their right customers and improve sales through the use of machine learning algorithms. The system will use various algorithms to analyze customer behaviour and preferences and generate personalized recommendations for each customer.

    • Objectives:
      • The primary objective of this project is to enable businesses to sell their products online and reach a wider audience through the development of an e-commerce website.
      • The website will be powered by a machine learning algorithm that will generate personalized product recommendations for customers,thereby improving their shopping experience.
      • The project will involve the development and hosting of an e-commerce website that can handle large volume of traffic,process transaction securely,and store customer data in a reliable and efficient mannder.
    • Front-end and Back-end used:
      • HTML,CSS,BOOTSTRAP,Javascript for the front-end
      • MySQL for the database, and Flask framework of Python for the back-end
    • Tools and Technologies Used:
      • Python Programming Language
      • PIP
      • Flask Framework
      • HTML5
      • CSS3
      • BootStrap 4.2
      • JavaScript
      • MySQL Database
      • Wamp Server
